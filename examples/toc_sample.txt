# 目次機能のサンプル

:::目次:::

:::見出し1
第1章: 序論
:::

このドキュメントは目次機能のデモンストレーションです。
左側のサイドバーに目次が表示され、クリックすると該当箇所にジャンプします。

:::見出し2
1.1 背景
:::

長い文書では目次があると便利です。読者は全体の構造を把握しやすくなり、
必要な情報に素早くアクセスできます。

:::見出し3
1.1.1 従来の問題点
:::

- 手動で目次を作成する必要があった
- 章番号の管理が煩雑
- ページ内リンクの設定が面倒

:::見出し3
1.1.2 解決策
:::

Kumihan-Formatterの目次機能により、これらの問題が解決されます。

:::見出し2
1.2 機能概要
:::

目次機能の主な特徴：

- 自動的に見出しから目次を生成
- ページ内リンクで素早くナビゲーション
- 表示/非表示の切り替え可能
- 現在位置のハイライト表示

:::見出し1
第2章: 実装方法
:::

:::見出し2
2.1 基本的な使い方
:::

目次を表示するには、文書の先頭付近に以下の記法を追加します：

```
:::目次:::
```

これだけで、自動的に見出しから目次が生成されます。

:::見出し2
2.2 高度な使い方
:::

:::見出し3
2.2.1 複合見出し
:::

複合キーワードを使った見出しも目次に含まれます：

:::見出し3+太字
2.2.2 重要なセクション
:::

このように太字と組み合わせた見出しも、正しく目次に表示されます。

:::見出し2
2.3 注意事項
:::

- モバイル環境では目次は表示されません
- 16:9のPC環境での利用を想定しています
- 印刷時は目次は非表示になります

:::見出し1
第3章: まとめ
:::

目次機能により、長い文書の読みやすさが大幅に向上します。
特にシナリオのような章立て構成の文書では必須の機能と言えるでしょう。

:::見出し2
3.1 今後の展望
:::

将来的には以下の機能拡張も検討されています：

- 目次の階層レベル指定
- 番号付き目次オプション
- カスタムスタイルの適用
