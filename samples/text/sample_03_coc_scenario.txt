# 枠線 #
**シナリオタイプ**: クローズド・探索
**推奨人数**: 3-4人
**推定時間**: 4-6時間
**舞台**: 1920年代、アーカム郊外の古い館
##

# 見出し3 #
導入
##

探索者たちは古い友人から一通の手紙を受け取る。手紙の主は故エドワード・ウィンスロップ氏の遺産整理を依頼されている弁護士で、探索者たちにウィンスロップ邸の調査を依頼したいという内容だった。

# 引用 #
「親愛なる諸君へ

故エドワード・ウィンスロップ氏の遺産整理に際し、同氏の所有していた館の調査をお願いしたく、筆を取った次第です。館は長らく無人であり、地元の人々は近づくことを避けています。

報酬として一人あたり200ドルをお支払いいたします。詳細については同封の地図をご参照ください。

                               弁護士 ジョナサン・クレイ」
##

# 太字 #
最初の印象
##

19世紀末に建てられた3階建ての大きな館。長年の風雨にさらされ、外壁には蔦が這い、屋根瓦の一部が落ちている。正門の鉄柵は錆び付いているが、まだ開閉は可能だ。

# 注意 #
**知覚ロール（目標値：60）**
- 成功：2階の窓の一つに人影のようなものが見える
- 失敗：特に異常は感じない
##

# 太字 #
儀式の阻止
##

探索者たちが地下室で『ネクロノミコン』を発見すると、本は自動的に開き始める。文字が光り、空間に歪みが生じる。

# 注意 #
**緊急事態！**
- 儀式を阻止するには【オカルト】ロール（目標値：30）
- 失敗すると次元の門が開き始める
- 3ラウンド以内に阻止できなければ...
##

# 折りたたみ #
**最悪の結末**

門が完全に開くと、R'lyehから这い出してきた怪物たちが現実世界に侵入する。探索者たちは永続的なSAN値ロス（2d10）を受け、シナリオは破滅的な結末を迎える。

# 見出し2 #
ハンドアウト
##

# 枠線 #
**探索者全員へ**

あなたたちは弁護士ジョナサン・クレイから依頼を受け、故エドワード・ウィンスロップ氏の館を調査することになった。報酬は一人200ドル。館は長らく無人で、地元の人々は近づくことを避けているという。

**調査の目的**: 館の状況確認と遺品の整理
**期限**: 特になし
**注意事項**: 何か異常があれば即座に報告すること
##

# 太字 #
シナリオの所要時間: 約4-6時間
##

このシナリオは探索者たちの行動によって展開が変わります。柔軟に対応し、プレイヤー全員が楽しめるよう心がけてください。
