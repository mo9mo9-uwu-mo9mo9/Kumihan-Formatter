# Kumihan-Formatter インライン記法改善サンプル

## 基本的なインライン記法（終了マーカー##必須）

通常の文章に # 太字 # 重要な内容 ## を埋め込むことができます。

また、# イタリック # 補足情報 ## や # 下線 # 強調したい部分 ## も使用可能です。

## 色属性付きインライン記法

# 太字 color=red # 赤い太字テキスト ## や # イタリック color=blue # 青い斜体テキスト ## が可能です。

# 下線 color=green # 緑色の下線付きテキスト ## と # ハイライト color=yellow # 黄色背景のハイライト ## も対応。

## 複数のインライン記法の連続使用

この段落では # 太字 # 最初の強調 ## と # イタリック # 次の強調 ## と # 下線 # 最後の強調 ## を連続して使用しています。

## リスト内でのインライン記法

# リスト #
# 太字 # 重要な項目1 ##
# イタリック # 補足的な項目2 ##
 # 下線 # ネストされた子項目 ##
 # ハイライト color=yellow # 黄色でハイライトされた子項目 ##
通常の項目3
##

## ブロック内でのインライン記法

# 枠線 #
この枠線ブロック内で # 太字 # 強調テキスト ## を使用しています。
また、# ハイライト color=cyan # シアン色のハイライト ## も可能です。
##

# 注意 #
⚠️ # 太字 color=red # 重要な注意事項 ## ：システムに影響があります。
詳細は # イタリック # マニュアルを参照 ## してください。
##

## 複雑な組み合わせ

# 情報 #
# 太字 # 新機能リリース ## について：
- # イタリック # バージョン2.1.0 ## から利用可能
- # 下線 # ドキュメント更新済み ##
- # ハイライト color=lime # パフォーマンス改善 ## を実現
##

## 特殊なケース

### コード記法
インラインで # コード # print("Hello") ## を記述できます。

### 取り消し線
# 取り消し線 # 削除された内容 ## もサポートしています。

### エラーケース（終了マーカーなし）
以下は正しく処理されません：
# 太字 # 終了マーカーがない

### 正しい記法
# 太字 # 必ず終了マーカー ## が必要です。
