# 情報 #
## 🔄 正規化テストの重要性

文字の見た目は同じでも、内部的には異なる表現の可能性：
- # コード # U+00E9 ## (単一文字のé) 
- # コード # U+0065 + U+0301 ## (e + 結合アクセント)

システムは両方を同等に処理する必要があります。
