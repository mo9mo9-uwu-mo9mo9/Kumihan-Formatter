## 📌 基本の記法（これだけ覚えればOK！）

### 1. 見出し

#### 📝 Kumihan記法
```
;;;見出し1
第1章 はじめに
;;;

;;;見出し2
1.1 概要
;;;

;;;見出し3
1.1.1 詳細説明
;;;
```

#### 🖥️ 表示結果
> # 第1章 はじめに
> ## 1.1 概要
> ### 1.1.1 詳細説明

---

### 2. 太字（強調）

#### 📝 Kumihan記法
```
;;;太字
重要：これは必ず読んでください
;;;
```

#### 🖥️ 表示結果
> **重要：これは必ず読んでください**

---

### 3. 枠線

#### 📝 Kumihan記法
```
;;;枠線
NPCの特徴：
- 身長180cm
- 黒髪で眼鏡をかけている
- 物静かな性格
;;;
```

#### 🖥️ 表示結果
> ┌─────────────────────┐
> │ NPCの特徴：         │
> │ - 身長180cm         │
> │ - 黒髪で眼鏡をかけている │
> │ - 物静かな性格        │
> └─────────────────────┘

## 🎨 装飾系の記法

### 7. ハイライト（背景色付き）

#### 📝 Kumihan記法
```
;;;ハイライト
SANチェック：1d10/1d100
;;;

;;;ハイライト color=#ffe6e6
危険！この部屋には入らないこと
;;;

;;;ハイライト color=#e6ffe6
安全：ここで休憩できます
;;;
```

#### 🖥️ 表示結果
> 🟨 SANチェック：1d10/1d100

> 🟥 危険！この部屋には入らないこと

> 🟩 安全：ここで休憩できます

---

## 🎯 複合記法（上級者向け）

### 12. 複数のスタイルを組み合わせる

#### 📝 Kumihan記法
```
;;;太字+枠線
重要な手がかり：日記の最後のページ
;;;

;;;見出し2+太字
第2章 事件の真相
;;;

;;;枠線+ハイライト color=#fff3cd
ヒント：鍵は書斎の本棚にある
;;;
```

#### 🖥️ 表示結果
> ┌─────────────────────────┐
> │ **重要な手がかり：日記の最後のページ** │
> └─────────────────────────┘

> ## **第2章 事件の真相**

> 🟨┌────────────────────┐
> 　│ ヒント：鍵は書斎の本棚にある │
> 　└────────────────────┘

## 📑 段落と改行

### 13. 段落の区切り

#### 📝 Kumihan記法
```
これは最初の段落です。
同じ段落内の文章は続けて表示されます。

空行を入れると新しい段落になります。
これで見やすい文章構成が作れます。
```

#### 🖥️ 表示結果
> これは最初の段落です。同じ段落内の文章は続けて表示されます。
>
> 空行を入れると新しい段落になります。これで見やすい文章構成が作れます。

## 💡 便利なテンプレート集

### NPCテンプレート

```
;;;見出し2
NPC：山田太郎
;;;

;;;枠線
基本情報：
- 年齢：35歳
- 職業：私立探偵
- 特徴：鋭い観察眼を持つ
;;;

;;;太字
重要な情報
;;;
彼は事件の重要な証人である。

;;;ネタバレ
実は犯人を目撃していた
;;;
```

### アイテムテンプレート

```
;;;見出し3+太字
古い日記
;;;

;;;枠線
アイテム詳細：
- 発見場所：書斎の引き出し
- 状態：ところどころページが破れている
- 重要度：★★★★☆
;;;

;;;ハイライト color=#ffe6e6
SANチェック：成功すると内容を読める
;;;
```

### 場所テンプレート

```
;;;見出し2
洋館・1階ホール
;;;

;;;枠線
部屋の特徴：
- 広さ：約50平米
- 照明：シャンデリア（故障中）
- 出入口：北、東、西の3箇所
;;;

調査可能な箇所：
1. 暖炉
2. 絵画（壁にかかっている）
3. 本棚
4. 絨毯（何かの跡がある）
```

## ⚠️ よくある間違い

### ❌ 間違った記法

```
# これはMarkdownの見出し（使えません）

**これもMarkdown**（使えません）

;;;太字
閉じ忘れ（エラーになります）
```

### ✅ 正しい記法

```
;;;見出し1
これがKumihanの見出し
;;;

;;;太字
これがKumihanの太字
;;;
```
