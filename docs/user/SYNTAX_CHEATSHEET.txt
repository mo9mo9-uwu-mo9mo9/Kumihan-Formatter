# Kumihan-Formatter 記法一覧
# ===========================
# テキストファイル(.txt)を美しいHTMLに変換するための記法集

## 段落と改行
----------------------------------------
通常のテキスト
連続する行は同じ段落になります

空行で段落を区切ります


## 見出し
----------------------------------------
;;;見出し1
大見出し（第1レベル）
;;;

;;;見出し2
中見出し（第2レベル）
;;;

;;;見出し3
小見出し（第3レベル）
;;;

;;;見出し4
より小さい見出し（第4レベル）
;;;

;;;見出し5
最小の見出し（第5レベル）
;;;


## 文字装飾
----------------------------------------
;;;太字
太字のテキスト
;;;

;;;イタリック
イタリック体のテキスト
;;;


## 枠線とハイライト
----------------------------------------
;;;枠線
枠線で囲まれたブロック
複数行も可能です
;;;

;;;ハイライト
デフォルト色（黄色）のハイライト
;;;

;;;ハイライト color=#ff0000
赤色のハイライト
;;;

;;;ハイライト color=#00ff00
緑色のハイライト
;;;

;;;ハイライト color=#0000ff
青色のハイライト
;;;


## リスト
----------------------------------------
箇条書きリスト:
- 項目1
- 項目2
- 項目3

番号付きリスト:
1. 最初の項目
2. 次の項目
3. 最後の項目


## キーワード付きリスト
----------------------------------------
- :太字: 太字の項目
- :イタリック: イタリックの項目
- :枠線: 枠線付きの項目
- :ハイライト: ハイライト付きの項目
- :ハイライト color=#ffd700: 色指定ハイライト付きの項目
- :見出し3: 見出しスタイルの項目


## 複合キーワード（+で連結）
----------------------------------------
;;;太字+イタリック
太字かつイタリックのテキスト
;;;

;;;枠線+太字
枠線で囲まれた太字テキスト
;;;

;;;見出し2+太字
太字の見出し（レベル2）
;;;

;;;枠線+ハイライト color=#ffff00
枠線と黄色ハイライトの組み合わせ
;;;

;;;見出し3+枠線+太字
3つの装飾を組み合わせた例
;;;


## 複合キーワード付きリスト
----------------------------------------
- :太字+枠線: 太字で枠線付きの項目
- :見出し3+太字: 見出しスタイルの太字項目
- :枠線+ハイライト color=#e6f3ff: 枠線と薄青ハイライトの項目
- :太字+イタリック+枠線: 3つの装飾を組み合わせた項目


## 画像埋め込み
----------------------------------------
;;;画像ファイル名.png;;;
;;;画像ファイル名.jpg;;;
;;;画像ファイル名.gif;;;

例:
;;;scenario_map.png;;;
;;;character.jpg;;;


## 目次
----------------------------------------
;;;目次;;;

※文書の先頭付近に配置すると、自動的に見出しから目次を生成
※PC環境でサイドバーに表示（16:9画面推奨）


## 記法の組み合わせ例
----------------------------------------
;;;枠線
【重要な情報】

;;;太字
ここは特に重要です
;;;

- :太字: 重要ポイント1
- :ハイライト color=#ffe6e6: 注意事項
;;;

;;;見出し2+枠線+ハイライト color=#f0f8ff
複雑な組み合わせの見出し
;;;


## エラー表示
----------------------------------------
不正な記法やキーワードは以下のように表示されます:

[ERROR: 無効なキーワード]
（背景色 #ffe6e6 で強調表示）


## 記法のルール
----------------------------------------
1. ブロックマーカーは ;;; で開始・終了
2. キーワードは半角コロン : で囲む
3. 複合キーワードは + または ＋ で連結
4. color属性は半角スペースの後に指定
5. リストは - または番号. で開始
6. 画像ファイル名は拡張子付きで指定

## コメント
----------------------------------------
# で始まる行はコメント（HTMLに出力されない）
# このようにメモを残せます
