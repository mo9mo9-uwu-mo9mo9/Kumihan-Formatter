段落と改行の違い
##

Kumihan-Formatterでは「改行したら、そのまま改行される」直感的な処理を採用しています。

# 太字 #

##

これは1行目です。
これは2行目です。
これは3行目です。

上記は1つの段落内で改行されており、HTML出力では<br>タグで改行されます。

# 太字 #
段落の区切り例（pタグ）
##

段落は空行で区切られます。これは最初の段落です。

これは2番目の段落です。複数行にわたって書くことができ、さまざまなブロック記法を使用してスタイルを適用することができます。

上記は2つの独立した段落として処理され、HTML出力では別々の<p>タグになります。

# 枠線 #
📝 改行と段落の使い分け

🔸 同じ話題を続ける場合：
  → 改行のみ（空行なし）
  → HTML: <br>タグで改行

🔸 話題が変わる場合：
  → 空行で段落を区切る
  → HTML: 新しい<p>タグ

この仕組みにより、Markdownのような複雑な記法を覚える必要がありません！
##

##

# ハイライト color=#e6f3ff #

- 改行: 行末に2つのスペース + 改行

Kumihan-Formatterの場合：
- 改行: そのまま改行するだけ ← 簡単！
- 段落: 空行で区切る（同じ）

# 見出し3 #
変換コマンド例
##

このファイルを変換するには：

# コード #
基本変換:
python -m kumihan_formatter.cli comprehensive-sample.txt

ダークテーマで変換:
python -m kumihan_formatter.cli comprehensive-sample.txt --config config-sample.yaml

ファイル監視モードで変換:
python -m kumihan_formatter.cli comprehensive-sample.txt --watch
