# 太字+ハイライト color=#f0f8ff #
NPC作成のポイント
##
1. 明確で覚えやすい特徴を持たせる
2. シナリオでの役割を明確にする
3. 探索者との関係性を考慮する
4. リアルで一貫性のある人格を与える
5. 適切な情報量で魅力的に描く
